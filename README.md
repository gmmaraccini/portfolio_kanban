# portfolio_kanban
Projeto 8 - Kanban

8. Quadro de Tarefas (Tipo Trello/Kanban)
O que faz: Um sistema onde o usuário pode criar "quadros", "listas" (Ex: A Fazer, Fazendo, Feito) e "cartões" (tarefas), e arrastá-los entre as listas.
Habilidades que demonstra: Estrutura de banco de dados mais complexa (relacionamentos), e provavelmente o uso de um framework PHP moderno.
